About the Project

This project focuses on detecting brain tumors from MRI images using Machine Learning and Deep Learning techniques. It assists doctors by providing fast and accurate tumor identification.

Types of Brain Tumors

Glioma

A tumor arising from glial cells in the brain.

Meningioma

A tumor that forms on membranes covering the brain.

Pituitary Tumor

A tumor occurring in the pituitary gland.

No Tumor

Healthy brain MRI with no abnormalities.

Machine Learning Model

The system uses Convolutional Neural Networks (CNNs) trained on MRI datasets. The model extracts features, classifies tumor types, and provides prediction results with high accuracy.
